**Adult App** is a student project built using **Next.js**, **TypeScript**, and **Tailwind CSS**.

The application is designed to help young adults learn basic life skills through **step-by-step guides** and **community support**.

## Overview

The prototype includes:

- User onboarding
- Life skill guides (budgeting, renting, taxes)
- Local information and resources
- Community discussions
- Search functionality
- User profile settings

## Purpose

The goal of this project is to demonstrate a **functional and user-friendly web application** that helps users navigate everyday adult responsibilities.
